Identifies which media library files are in use in WordPress content and allows you to delete unused ones.

== Description ==

Media Usage Checker is a powerful tool that helps you identify and manage your WordPress media library files. It scans your content to determine which media files are being used and which ones are not, allowing you to safely clean up your media library.

= Key Features =

* Scan your entire media library for usage
* Identify unused media files
* Bulk delete unused files
* Detailed usage statistics
* Background scanning to prevent server overload
* Support for various file types
* Integration with popular page builders

= File Type Support =

* Images (jpg, png, gif, etc.)
* Documents (pdf, doc, docx, etc.)
* Videos (mp4, mov, avi, etc.)
* Audio files (mp3, wav, ogg, etc.)

= Enhanced Interface =

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/media-usage-checker` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Media Usage Checker screen to configure the plugin
4. Start scanning your media library for unused files

== Frequently Asked Questions ==

= How does the plugin determine if a file is in use? =

The plugin checks for file usage in:
* Post and page content
* Custom post types
* Widgets
* Theme customizer settings
* Page builder content
* Custom fields
* And more

= Will deleting unused files affect my site? =

No, the plugin only allows deletion of files that are confirmed to be unused in your WordPress content.

= How often should I scan my media library? =

We recommend scanning your media library:
* After major content updates
* Before site backups
* Periodically (weekly or monthly)

= Does the plugin work with page builders? =

Yes, the plugin is compatible with popular page builders including:
* Elementor
* Divi
* WPBakery
* Beaver Builder
* And others
