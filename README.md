# Media Usage Checker - WordPress Plugin

## Description
Media Usage Checker is a WordPress plugin that helps administrators and content managers identify and manage media file usage across their website. This tool provides a comprehensive overview of which media files are being used in posts and pages, and allows for easy deletion of unused files directly from the WordPress media library.

## Features
- Scan your entire WordPress site for media usage
- Identify unused media files
- Track media usage in:
  - Posts
  - Pages
  - Custom post types
- Generate detailed reports of media usage and unused files
- Direct deletion of unused media files from within the plugin interface
- Bulk actions for unused media deletion
- User-friendly interface
- Performance-optimized scanning

## Installation
1. Download the plugin ZIP file.
2. Log in to your WordPress admin panel.
3. Go to Plugins > Add New.
4. Click on Upload Plugin.
5. Choose the ZIP file and click Install Now.
6. After installation, click Activate.

## Usage
### Basic Usage
1. Go to Media > Media Usage Checker in your WordPress admin panel.
2. The plugin will automatically scan your media library and content.
3. View the results, which are separated into "Files in Use" and "Unused Files".
4. For unused files, you can delete them individually or in bulk directly from this interface.

### Advanced Features
- Bulk Actions: Select multiple unused media files for deletion.
- Pagination: Navigate through large lists of media files easily.
- File Size Display: See the size of each media file to help with storage management.

## Requirements
- WordPress 5.0 or higher
- PHP 7.2 or higher
- MySQL 5.6 or higher

## Frequently Asked Questions
Q: Is it safe to delete unused media files?
A: While the plugin accurately tracks media usage, we recommend creating a backup before deleting any files. Once deleted, files cannot be recovered without a backup.

Q: Will this plugin slow down my website?
A: No, the scanning process is optimized and doesn't affect your site's front-end performance.

Q: Can I recover deleted files?
A: No, this plugin deletes files permanently. Always ensure you have a backup before deleting files.
